Projet de développement JAVA

Situation : Une tempête magique a déplacé des cristaux dans un univers fictif, et il faut les réaligner dans leurs temples respectifs.

Tâche : Développer une application interactive permettant d'optimiser les déplacements des cristaux à l'aide d'algorithmes de tri et heuristiques.

Action :

  • Implémentation d'un algorithme de tri par sélection pour échanger et placer les cristaux.
  • Développement d'un algorithme heuristique ciblant en priorité les cristaux mal placés pour optimiser leur réalignement.
  • Création d'une interface interactive permettant aux utilisateurs de déplacer un apprenti pour collecter et déposer les cristaux.

Résultat: Obtention d'une excellente note permettant la validation du module.

Développement d'un site Web

Situation : Dans le cadre d'un projet académique, il était nécessaire de concevoir un site web permettant de calculer des probabilités associées à la loi inverse-gaussienne.

Tâche : Développer un site web interactif capable d'effectuer ces calculs à l'aide de méthodes numériques et d'afficher les résultats sous forme graphique.

Action :

  • Implémentation des formules de calcul basées sur la loi inverse-gaussienne.
  • Développement d'une interface utilisateur intuitive pour saisir les données et visualiser les résultats.
  • Optimisation du code et gestion des performances pour assurer un temps de réponse rapide.

Résultat : Un site web fonctionnel permettant aux utilisateurs d'effectuer facilement des calculs de probabilités et de visualiser les résultats, développé en collaboration avec une équipe de quatre personnes en 200 heures de travail.
